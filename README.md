# BlackSpace <img src="./BlackSpace.mp4" alt="BlackSpace logo" width="50">

BlackSpace 是 BlackLab 旗下的個人財富管理工具，將日常記帳、投資組合、股票資訊、投資新聞、預算管理和模擬交易集中在同一個 Web App 內。目標是讓用戶可以用一個介面管理現金流、了解資產狀況，並輔助日常投資觀察。

## 主要功能

### Dashboard 總覽

- 顯示總支出、平均每月支出、銀行存款、投資總值和總資產。
- 支援年度或自訂月份範圍的收入、支出、投資、現金流和儲蓄率摘要。
- 提供分類支出、月度趨勢和財務洞察。
- 支援私隱模式，開啟後 Dashboard 的財產金額會以 `*****` 顯示。

### Monthly View 每月收支

- 按月份查看收入、支出、淨儲蓄、交易數量和平均交易金額。
- 可新增、編輯和刪除交易。
- 顯示各分類支出分佈。
- 提供 Budget vs Actual，方便比較預算與實際支出。
- 可設定每月收入和查看不同月份的現金流變化。

### Categories 分類管理

- 建立和管理支出分類。
- 每個分類可設定名稱、顏色和每月預算。
- 支援固定每月支出設定，用於自動建立重複交易。
- 可按分類追蹤支出次數和預算使用情況。

### Goals 儲蓄目標

- 建立不同儲蓄或財務目標。
- 設定目標金額和目前已儲蓄金額。
- 以進度條顯示完成百分比。
- 支援編輯和刪除目標。

### Investments 投資組合

- 新增和管理股票、ETF、加密資產等持倉。
- 顯示成本總額、市值、總盈虧、回報率、資本、剩餘現金和帳戶總值。
- Capital 和 Cash Remaining 由用戶自行輸入。
- 支援不同貨幣持倉。
- 提供投資組合分佈圖，查看現金、股票、加密資產等配置比例。

### Investment News 投資新聞

- 可輸入股票代號查看相關新聞。
- 支援熱門股票快捷選擇。
- 可按最新、情緒或相關度排序新聞。
- 顯示新聞來源、時間、摘要、相關股票代號和情緒標籤。
- 提供新聞重點與整體影響分析，協助快速理解市場消息。

### Stock Research 股票研究

- 輸入股票代號後查看公司和股價資料。
- 顯示財務快照、估值指標、財務趨勢和同業比較。
- 提供風險與新聞監察。
- 提供公司整體分析、財務健康度和投資評價參考。
- 支援分批建倉價格節奏，包括 1/2 和 1/4 倉位安排。

### Sim Trading 模擬交易

- 進行模擬股票買賣。
- 支援觀察清單、股票搜尋、買入、賣出和持倉管理。
- 顯示現金、持倉市值、總資產、盈虧和回報。
- 支援訂單記錄和待成交訂單查看。

### 設定與資料管理

- 支援語言切換。
- 支援資料匯出和匯入。
- 支援登入和註冊帳戶。

### 手機與 Web App 體驗

- 針對手機版和 iOS Web App 進行介面調整。
- 底部工具列支援主要功能快速切換。
- 介面會配合手機安全區域和鍵盤彈出情況作顯示處理。

## 產品定位

BlackSpace 不是單純記帳工具，而是一個結合個人財務管理和市場資訊的多功能 App。它適合想同時管理日常收支、追蹤資產、查看投資新聞、研究股票和進行模擬交易的用戶。

## 免責聲明

BlackSpace 所提供的投資新聞、股票研究和模擬交易功能僅供學習及參考用途，並不構成任何投資建議。投資涉及風險，包括可能損失本金。用戶應自行判斷並承擔所有投資決策責任。
